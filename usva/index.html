Installation Requirements

You need at least WordPress version 4.8+ installed for this plugin to work properly. It is strongly recommended that you always use the latest stable version of WordPress to ensure all known bugs and security issues are fixed. Other requirements:

  • PHP version 5.3 or greater.
  • MySQL version 5.0.15 or greater.
  • WPBakery Page Builder 6.5.0 or greater.

Installation Steps

To install a plugin, you must first upload the plugin files and then activate the plugin. The plugin files can be uploaded in two ways:

  • FTP Upload: Using your FTP client, upload the non-zipped plugin folder into the /wp-content/plugins/ folder on your server.
  • WordPress Upload: Navigate to Plugins >> Add New >> Upload. Go to browse, and select the zipped plugin folder. Hit "Install Now" and the plugin will be uploaded and installed.

Step 01: Go to WP admin dashboard and click on Plugins >> Add New.

Step 02: Click on Upload Plugin and upload 'ultimate-searchable-accordion-vc-addon.zip' file. Finally, click Install Now button.

Step 03: Once plugin successfully uploaded in the server, you will get a message to activate it. Click on Activate Plugin Link and plugin will be ready to use . After activating the plugin, you will be redirected in plugins page of the WordPress admin panel, and you will get newly installed Ultimate Searchable Accordion - WPBakery Page Builder Addon in there.

Step 04: Now you are ready to use Ultimate Searchable Accordion - WPBakery Page Builder Addon!

General Tab:

This tab allows you to set the common settings of the ultimate searchable accordion addon. We are going to discuss detail about each option in this tab in below. Here goes the General Tab screenshot.

  • Enable Accordion Toggle Status: By default accordion blocks open and closes individually. That means, there is no dependencies between the accordion blocks. Howeever, if you would like to close an accordion detail block while clicking on the other accordion block title then check Enable Accordion Toggle Status checkbox.
  • Enable Expand Collapse Button: The Addon allows you to display an eye-catching Expand/Collapse All button box top of the accordion container. By default, Addon does not displays that control box. To show this button box, just check Enable expand/collapse button checkbox. You will get an output like the following screenshot.
  • Disable Closed All Section: This option contains a sub option feature. Specifically, when you click on the checkbox, another dropdown box will appear just below it. Four options will be available there. You can either keep all the rows opened or keep only even rows opened or keep only odd rows opened or keep custom rows opened by assigning the position of rows.

  • Enable RTL Mode: The Addon provides perfect support for RTL (Right To Left) website. Just check Enable RTL Mode checkbox, and the addon will automatically adjust the text and other content position.
  • Accordion Extra Class: The addon allows you to add custom classes for the accordion block. That way, we do not need to change any CSS code inside the addon.

Plugin Translation

Ultimate Searchable Accordion - WPBakery Page Builder Addon WordPress Plugin offers cool translation feature. So, beside the English you can easily translate the theme text for other languages as well. Check following steps to translate "Ultimate Searchable Accordion - WPBakery Page Builder Addon".

Step 01: To translate the plugin texts, please download and install Poedit software in your computer. Here goes the download link- https://poedit.net/download

Step 02: Next, navigate to /wp-content/plugins/ultimate-searchable-accordion-vc-addon/lang/. You will get a file named default.po.

Step 03: Our Plugin text-domain is 'usa_vc'. Now, we want to translate our theme text in to Dutch language. So, copy default.po file and rename it as usa_vc-de_DE.po

Step 04: Now, open that newly created 'usa_vc-de_DE.po' file using Poedit software. You will get all the plugin texts in there and you just need to add appropriate translated text in Translation box.

Step 05: Finally, save the file and you will get 'usa_vc-de_DE.mo' file inside languages folder.
